### double类型的List排序

```java
    List list = new ArrayList<>();
    list.add(new Double(93));
    list.add(new Double(53));
    list.add(new Double(79));
    list.add(new Double(19));
    list.add(new Double(25));
    list.add(new Double(57));
    list.add(new Double(76));
    list.add(new Double(8));

    Collections.sort(list);
```
